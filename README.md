# AI Resume Screening System

## Overview
An AI-powered Resume Screening System built using Python, Streamlit, and Machine Learning.

## Features
- Resume text input
- Job category prediction
- Machine Learning classification
- Streamlit web interface

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
